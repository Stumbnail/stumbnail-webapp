# Assets Directory

This directory contains image assets for the Stumbnail application.

## Directory Structure

### `/thumbnails`
Place thumbnail images here (15 images recommended for the login page background grid):
- `thumb1.jpg` through `thumb15.jpg`
- Recommended size: 280x160px
- Format: JPG or PNG

### `/avatars`
Place user avatar images here (4 images for the trust indicator):
- `user1.jpg` through `user4.jpg`
- Recommended size: 80x80px (circular)
- Format: JPG or PNG

## Downloading Assets from Figma

1. Open the Figma design file
2. Select each image asset
3. Export at 2x or 3x resolution for better quality
4. Save to the appropriate directory
5. Rename files to match the naming convention above

## Fallback

If images are not provided, the design will show gray placeholder backgrounds while maintaining the layout and functionality.
